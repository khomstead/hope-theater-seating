Custom seating chart system for HOPE Theater venues with WooCommerce/FooEvents integration.

== Description ==

HOPE Theater Seating provides visual seat selection for theater venues with full WooCommerce and FooEvents integration.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/hope-theater-seating/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure venue settings in the admin panel
